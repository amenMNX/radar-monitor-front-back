# `src` Folder

This folder contains all source code for the Spring Boot backend.

It follows the standard Maven project structure:

- `main`: application code and runtime configuration.
- `test`: automated tests.

Spring Boot and Maven expect this structure by convention. Keeping the backend organized this way makes it easier for Maven, IDEs, and Spring Boot tools to compile, run, package, and test the project.

In this backend, `src` is the entry point for everything related to the Java application itself. The `pom.xml` file at the backend root describes dependencies and build settings, while this folder contains the code that those dependencies are used by.
